## 功能:
* 1. 在8x8的LED全彩點矩陣上，隨機生成蛇的食物(會避開蛇的身體位置)
* 2. 做到碰撞偵測(碰到牆壁與自己的身體則遊戲結束)
* 3. 隨著吃到的食物數量越多，蛇的身體長度會增長並加快蛇的前進速度
* 4. 用七段顯示器將分數展現出來
